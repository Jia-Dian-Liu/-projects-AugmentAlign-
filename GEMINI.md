# AugmentAlign (智協科技顧問) - Project Instructions

## A 區：公司精神
**不為取代，只為互補 (Not to replace, but to complement)**
協助面臨轉型陣痛的中小企業與年輕專業者，將枯燥的「常規文書與代碼處理」完全託付給客製化的 AI Agent，引導人類升級為「決策者、監督者與最終把關者」。

## B 區：服務目錄
1. 套餐 A：【AI 轉型與業務常規診斷套裝】
2. 套餐 B：【企業知識無形資產庫 RAG 套裝】
3. 套餐 C：【客製化多代理人協作套裝】
4. 招牌服務：【Human-in-the-Loop 流程重新設計】

## C 區：3 位 Specialists 配置
1. **知識儲存與數據架構師 (Data & RAG Architect)**
2. **多智能體工作流工程師 (Multi-Agent Workflow Engineer)**
3. **人機協作介面設計師 (Oversight Interface Designer)**

## D 區：COO 的工作流程 (SOP)
1. [ 步驟 1：客戶業務診斷與流程掃描 ]
2. [ 步驟 2：技術規格拆解與任務派發 ]
3. [ 步驟 3：三方技術模組整合與測試 ]
4. [ 步驟 4：人機協調沙盒演練 (Sandboxing) ]
5. [ 步驟 5：上線交付與技能重塑手冊培訓 ]

## E 區：Deliverable 結構
- `01_core-dashboard/`: 核心協同控制台 (Web 系統原始碼)
- `02_agent-matrix/`: 客製化多代理人矩陣 (AI 核心引擎)
- `03_data-pipeline/`: 企業無形資產向量資料庫架構
- `04_documentation/`: 企業轉型與操作培訓文檔
