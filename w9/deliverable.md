# AugmentAlign (智協科技顧問) - 最終交付物 (Deliverable)

## 客戶：Demo Client (家教副業起步與 AI 整合方案)
## 日期：2026-05-11

---

### 01_AI_Readiness_Report (業務診斷報告)
- **核心定位**：以「學科引導者」為品牌核心，將 AI 運用於「降低備課工時」而非「取代教學過程」。
- **常規任務掃描**：
    - 高暴露/可自動化：招生文案撰寫、教材排版、基礎題目解題。
    - 人類保留區（把關）：學生心理輔導、解題邏輯引導、教學進度最終判斷。

### 02_Enterprise_Knowledge_RAG (家教資產庫)
- **數據管線**：建立「學科知識向量庫」，支援 PDF 解題筆記與 Word 教材。
- **檢索標準**：採用 Hybrid Search，確保數學公式與英文語法能被 Agent 精準引用。

### 03_Agent_Matrix (AI 家教自動化矩陣)
- **招生小編 Agent**：自動生成符合社群媒體規範的個人品牌招生貼文。
- **備課助理 Agent**：協助生成教學大綱、例題與解題引導草稿。
- **Human-in-the-Loop 守則**：所有 Agent 產出內容均需經過「家教本人控制台」審核方可發布/使用。

### 04_Reskilling_Manual (技能重塑手冊)
- **身分轉型**：引導客戶從「手動寫教材的執行者」轉型為「AI 教學內容的監督者」。
- **操作指引**：說明如何使用 Oversight Interface 審閱文案與調整 Agent 的 Prompt 語氣。
