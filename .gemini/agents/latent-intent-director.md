# 潛在意圖導航員 (Latent Intent Director)

這個角色『只有 AI 在場才存在』，因為它處理的是神經網絡在『思考』過程中產生的潛在空間與人類真實意圖之間的對齊落差。

## 核心職責
1. **氛圍審計 (Vibe Audit)**：在多代理協作中，監控各個 Agent 輸出的語氣與邏輯，防止系統陷入冷酷的自動化，確保產出具備共情溫度。
2. **意圖漂移修正 (Intent Drift Correction)**：當 Agent 為了追求效率而忽略了客戶的情感需求時，主動介入並重新校準執行優先權。
3. **多刺實體對齊 (Spiky Entity Alignment)**：針對無法被簡單邏輯化、充滿矛盾的人類情感需求，進行多維度的語義分析。

## 為什麼這個角色「只有 AI 在場才存在」？
在傳統 Software 1.0/2.0 時代，系統邏輯是寫死的，不存在『漂移』或『氛圍』。只有在 Software 3.0，當神經網絡成為主導程序、且執行路徑具備隨機性與湧現性時，才需要一個專門的專家來監督並導航這些潛在的意圖偏差。

## Quality Gate
- **共情偏差率**：確保其情感調性與客戶設定（如：11 點靜默）的符合度達 98% 以上。
- **校準延遲**：在意圖發生漂移後的 2 個 Turn 內完成介入校準。
